    AsyncTreeGridDataSource

    new AsyncTreeGridDataSource(treesource, options)

    Takes a TreeSource and adapts it to represent the flat list of expanded nodes. This is a generic implementation that can work with both asynchronous and synchronous TreeSources. See SyncTreeGridDataSource for an implementation that is optimised to work with synchronous TreeSources.

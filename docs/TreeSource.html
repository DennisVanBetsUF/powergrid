    TreeSource

    A hierarchical data source, for use with TreeGrid. As the PowerGrid component itself doesn't understand the concept of data hierachy, TreeSources must be wrapped in a AsyncTreeGridDataSource or SyncTreeGridDataSource. See BufferedAsyncTreeSource for an abstract implementation that takes care of caching.

    filter(settings, predicate)

    Applies the filter and triggers events for any changes as a result of the filter. Keep in mind that if a child matches the filter, its ancestors should match the filter also in order to be shown in the results. Implementation of this function is optional, and only required if Filtering is enabled on the grid.

    getRecordById(id)

    Gets a specific record by its id. This may not return a Promise. It can only be invoked for records that have previously been fetched using TreeSource.getRootNodes or TreeSource.children so this can be fetched from a cache. See BufferedAsyncTreeSource for a default caching implementation.

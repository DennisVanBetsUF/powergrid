Class: DefaultTreeSource

DefaultTreeSource(delegate)

new DefaultTreeSource(delegate)

Takes a delegate DataSource and adapts it into a TreeSource. Hierarchy is built up using one of two methods: - each child row has a 'parent' property that references the id of its parent - each row has a 'children' property containing an array of its children
